Cunyfirst-api

an API to get classes from cunyfirst

View on GitHub Download .zip Download .tar.gz

This is a page I will be using to update users on my site No Closed Class.

The website can be used by students to get texted when a class opens up. There is also a course scheduler component to help students schedule calsses.

I am also looking to create a sharing resource for students to anonymously (or publicly if they care) share their tests, homework and class material with current and/or future students. This will help students without the connections over previous classmates be on a similar footing as the rest of their class.
